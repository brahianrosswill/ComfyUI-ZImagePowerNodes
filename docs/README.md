## Getting Started

To install **Z-Image Power Nodes** in your ComfyUI setup:

1. Open **ComfyUI Manager**.
2. Click on **Custom Nodes Manager**.
3. Search for `Z-Image Power Nodes`.
4. Click **Install** and restart ComfyUI.
