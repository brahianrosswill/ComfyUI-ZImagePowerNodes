# Workflows Directory

This directory contains example ComfyUI workflows demonstrating the usage and capabilities of the Power Nodes. The workflows are organized into two main categories based on their stability and development stage:

## Directory Structure

*   **📂 `/release`**
    *   **Description:** Contains stable, verified, and production-ready workflows.
    *   **Nodes Used:** Features finalized and thoroughly tested nodes.
    *   **Distribution:** These workflows are officially included in each production release package.

*   **📂 `/experimental`**
    *   **Description:** Contains work-in-progress (WIP), proof-of-concept, and highly experimental workflows.
    *   **Nodes Used:** May utilize features or nodes currently under testing.
    *   **Behavior:** These workflows are not fully optimized and might fail or produce unexpected results.
    *   **Distribution:** Excluded from the final release package; intended for development and testing only.
